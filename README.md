# LLM Text Preprocessing - Chapter 2

This repository contains an exercise based on Chapter 2 of *Build a Large Language Model (From Scratch)*.

## Main Files

- `embeddings.ipynb`: final notebook with core code, explanations, and the `max_length`/`stride` experiment.
- `ch02.ipynb`: original notebook downloaded from the book repository.
- `ch02.executed.ipynb`: fully executed version of the original notebook.
- `the-verdict.txt`: text file used for tokenization and sampling.

## Run

```powershell
py -m jupyter notebook
```
